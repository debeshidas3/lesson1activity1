# Lesson 1 Activity 1

## Purpose
This repository is intended for a Python project that demonstrates basic programming concepts, particularly focusing on data structures and algorithms. The activities aim to enhance the understanding of Python syntax and coding best practices.

## Structure
The repository is organized as follows:

```
lesson1activity1/
├── src/
│   ├── __init__.py
│   └── main.py
├── tests/
│   ├── __init__.py
│   └── test_main.py
└── README.md
```

- `src/` : Contains the source code of the project.
- `tests/` : Contains unit tests for the source code.

## Setup Instructions
1. **Clone the repository** to your local machine:
   ```bash
   git clone https://github.com/debeshidas3/lesson1activity1.git
   cd lesson1activity1
   ```
2. **Set up a virtual environment** (optional but recommended):
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. **Install the dependencies** (if any):
   ```bash
   pip install -r requirements.txt
   ```

## Usage Examples
To execute the main script:
```bash
python src/main.py
```

To run tests:
```bash
python -m unittest discover tests/
```
